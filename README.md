## README
- This repository contains the jupyter notebook and dataset used for the main project.
- To run this notebook as as the paths & directory names need to remain the same.
- For this to run it needs to be inside a virtual environment (`venv`) and the following packages need to be installed:
  - System:
    - `python3`
    - `git`
    - `nodejs` & `npm`

  - Python packages:
    - `jupyter-git` extension
    - `jupyter`, `pandas`, `scikit-learn`, `numpy`, `matplotlib`, `pillow` can be installed at once with the command:
      - `pip install pandas sklearn scikit-learn numpy matplotlib pillow jupyter`
